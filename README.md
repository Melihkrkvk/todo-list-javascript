# Todo List Javascript
## Patika project with JS

### As you see in the photos our feature are
- We can't add blanks to the list. And we get toast notification

- When we add a task to the list, we still get a toast notification.

- We do not lose our data because the added tasks are stored in localstorage.

- We can mark the tasks we have done by clicking them.

- We can delete our entire list with the listeyi temizle button.
